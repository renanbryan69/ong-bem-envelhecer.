Objetivo do Sistema: Desenvolver um sistema web para uma ONG de idosos que permita gerenciar informações sobre os idosos atendidos, voluntários, eventos, doações e atividades da instituição. O sistema também deve facilitar a comunicação com a comunidade, divulgar projetos, permitir o cadastro de voluntários e doadores e contribuir para uma administração mais eficiente, organizada e transparente.

 tecnologias utilizadas: O projeto foi desenvolvido utilizando HTML5, CSS3 e JavaScript. O HTML5 foi responsável pela estrutura e organização das páginas do site, o CSS3 foi utilizado para criar o layout, definir as cores, os estilos e a aparência da interface, enquanto o JavaScript adicionou interatividade, como o envio de formulários com mensagens de confirmação, o agendamento de doações e o destaque da página atual no menu de navegação. Trata-se de um site estático, desenvolvido apenas com tecnologias de front-end, sem a utilização de banco de dados, frameworks ou back-end.
